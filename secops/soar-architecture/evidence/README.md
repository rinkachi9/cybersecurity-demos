# Evidence: SOAR Architecture

## Expected Evidence

- Redacted SCC finding payload.
- Workflow execution result.
- Enrichment output with risk score.
- Remediation output in dry-run or test mode.
- Notification or ticket reference.
- Sample event and expected-result pair from `sample-events/` and `expected-results/`.

## Do Not Commit

- Real service account key IDs from production.
- External threat intelligence API keys.
- Slack, Jira, or ServiceNow tokens.
